Since numeric features have a large value range, we discretize them by mapping them into several intervals. For raisedHands, VisitedResources, AnnouncementsView and Discussion, we map [0, 100] into [0, 4]. For GradeID, we map [1, 12] into [1, 3]. This way, we eliminate the advantage numeric features have over category features in random forest.

Category Features

Since comparison between values of a category feature is meaningless, we use onehot vector to encode category feature. This way different categories is mapped into independent dimensions rather than one dimension.

We use pandas.get_dummies() to get onehot features.

xxxxxxxxxx
1
1
data = pandas.get_dummies(data, columns=config.discrete_features)

Take SectionID for example. SectionID's value range is {1, 2, 3}. After onehot encoding, SectionID is split into three dummy features, SectionID_1, SectionID_2, SectionID_3. Each dummy feature has only two value, 0 and 1.
